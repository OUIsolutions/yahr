Usage: yahl [OPTIONS]

Options:
  --p, --plotage <path>      Specify the directory or file for the plotage.
                            If not provided, a default path (.) will be used.
  --digest_mode <mode>      Set the digest mode for SHA behavior.
                            Options: "content" or "time".
                            - "time" will digest based on the modification time of the file or directory.
                            - "content" will digest based on the content.
  --d --delay <time>        Set the time to each reverification


Example:
  yahl -p /path/to/plotage --digest_mode content
  yahl -p /path/to/file.txt --digest_mode time
